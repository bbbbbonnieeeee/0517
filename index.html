二刀流-大谷翔平

個人資料

  • 原文姓名：おおたに しょうへい（Ohtani Shohei）
  • 綽號別稱：陸奧的達比修（みちのくのダルビッシュ）
  •                    真二刀流（リアル二刀流）、ShoTime
  • 出生日期：1994年07月05日（27歲）
  • 身高體重：193公分 102公斤
  • 投打習慣：右投左打
  • 守備位置：投手、外野手
  • 最快球速：165 km/h（2016年CS決勝輪第5戰） 
  •                    101 mph（美國職棒時期）
  • 擅長球路：指叉球、滑球、曲球
  • 出生地點：JPN.gif日本岩手縣水澤市（現：奧州市）
  • 最高學歷：JPN.gif日本花卷東高等學校
  • 職棒選秀：2012年日本職棒選秀會北海道日本火腿鬥士隊第一指名 
  •                    2017年美國職棒洛杉磯天使隊自由契約
  • 簽約金    ：1億日圓（約3000萬新台幣、火腿隊時期) 
  •                  231.5萬美金（約6945萬新台幣、洛杉磯天使隊時期）
  • 經紀人    ：CAA Sports
  • 推定年薪：550萬美金（2022年）
  • 親屬關係：父親為前社會人球隊球員大谷徹、母親為前羽球選手加代子
  •                    兄為前高知鬥犬隊球員大谷龍太、姐為大谷結香

經歷

  • 奧州市立水澤南中學校
  • 花卷東高等學校
  • 日本職棒北海道日本火腿鬥士隊（2013年～2017年）
  • 美國職棒洛杉磯天使隊（大聯盟，2018年～）

個人年表

  • 2011年 -- 夏季甲子園第一輪敗退
  • 2012年 -- 春季甲子園第一輪敗退
  • 2012年 -- 第二十五屆IBAF世界青棒錦標賽日本代表隊
  • 2012年 -- 07月19日，夏季甲子園岩手縣預賽中代表花卷東高等學校面對一關學院高等學校，於六局上兩出局二三壘有人時投出160km/h的速球將打者三振，刷新先前由佐藤由規創下的日本高校棒球史上最快球速紀錄
  • 2012年 -- 10月25日，大谷雖表明可能直接挑戰美國職棒，但北海道日本火腿鬥士隊仍在選秀會上以第一指名挑選他
  • 2012年 -- 12月04日，與北海道日本火腿鬥士隊監督栗山英樹為首的球團高層會面，決定到底是要直接挑戰大聯盟，或是從日職出發。而火腿球團也提出願意讓大谷投打兼修的養成計畫
  • 2012年 -- 12月09日，召開記者會正式接受火腿隊的第一指名，並對自己引起的騷動向火腿隊致歉。最後大谷以選秀球員最高規格的簽約金1億、年薪1500萬、激勵獎金5000萬日圓的合約加盟北海道日本火腿鬥士隊，球團方面也付予其早前赴美發展的王牌投手ダルビッシュ有的背號「11」以表重視
  • 2013年 -- 03月29日，對埼玉西武獅隊初登場，以先發第八棒‧右外野手身份上場守備。並於五局上對岸孝之擊出生涯首安、六局上擊出生涯首打點
  • 2013年 -- 05月23日，對東京養樂多燕子隊以先發投手身分初次登板，投5局失2分，無關勝敗
  • 2013年 -- 06月01日，對中日龍隊先發投5局失3分，獲得生涯首勝
  • 2013年 -- 07月10日，對東北樂天金鷲隊永井怜於四局上擊出生涯首轟
  • 2013年 -- 07月19日，以外野手身份首度參加明星賽，在第五局登板無失分後，從六局開始改守左外野。繼1996年鈴木一朗後再次有球員於同場明星賽中投球及打擊
  • 2014年 -- 05月13日，對埼玉西武獅隊達成生涯首次完投暨完封勝
  • 2014年 -- 07月19日，於明星賽中對央聯代表隊鳥谷敬於第一局投出162km/h速球，刷新Marc Kroon於2008年創下的161km/h日職明星賽最快球速紀錄
  • 2014年 -- 08月03日，對福岡軟銀鷹隊於第七局投出161km/h速球，追平佐藤由規於2010年創下的日職本土投手最快球速紀錄
  • 2014年 -- 08月06日，個人的明星賽球衣於慈善拍賣中以175.2萬日圓刷新自己去年的151.1萬紀錄，再次創下日職球衣義賣最高得標金額，款項也將作為東日本大地震的援助金
  • 2014年 -- 09月07日，對歐力士猛牛隊敲出本季第10發全壘打，達成日本職棒史上首度「10勝、10轟」紀錄，繼1918年美國職棒Babe Ruth、1982年韓國職棒金城漢後棒球史上第3人達成
  • 2014年 -- 10月05日，對東北樂天金鷲隊先發出賽調整，兩局的投球中，飆出4顆162km/h的速球，追平前讀賣巨人隊洋投Marc Kroon於2008年創下的日職最快球速紀錄，也刷新自己與佐藤由規的161km/h，成為日本投手史上最快球速紀錄
  • 2014年 -- 10月11日，於高潮賽首輪第1戰對歐力士猛牛隊先發主投6局奪勝，以20歲又3個月的年紀達成日職季後賽史上第三年輕勝投紀錄，僅次於1982年工藤公康的19歲5個月，以及2006年ダルビッシュ有的20歲1個月
  • 2014年 -- 美日野球對抗賽日本職棒明星隊
  • 2015年 -- 2015年第一屆世界12強棒球賽日本代表隊
  • 2015年 -- 12月04日，季後獲得球團加薪100%，由原本的年薪一億日幣加至兩億日幣，高中畢業第4年就獲得兩億年薪也追平ダルビッシュ有的日職紀錄
  • 2016年 -- 05月04日至05月17日，以打者身份達成連續5場出賽都擊出全壘打的紀錄
  • 2016年 -- 05月29日，在對東北樂天金鷲隊的比賽中，火腿隊取消指定打擊，讓大谷以先發投手‧第六棒的身分出賽，亦是大谷首度在洋聯例行賽中實現投球兼打擊的「二刀流」。該戰大谷投球7局6次三振失1分、打擊5打數3安打2得分1打點，贏得本季第3勝
  • 2016年 -- 06月05日，對讀賣巨人隊Luis Cruz於四局下投出163km/h的速球，刷新過去由Marc Kroon與自己共同保持的日職最快球速紀錄
  • 2016年 -- 07月03日，在對福岡軟銀鷹隊的比賽中首度以先發第一棒投手的「二刀流」身份出賽，並在開賽第一球對中田賢一擊出全壘打，成為日本職棒史上第一位擊出首局首打席全壘打的先發投手，繼2014年後再度達成單季10轟紀錄，該場大谷投8局10三振無失分，打擊2打數1安(全壘打)2得分1打點，終場火腿2:0勝，大谷獲得單季第8場勝投
  • 2016年 -- 07月15日，在明星賽第1天的全壘打大賽中，大谷先以6：5擊敗山田哲人，再以3：2擊敗柳田悠岐，首次參賽就獲得冠軍
  • 2016年 -- 07月16日，在明星賽第二場正賽中，打擊繳出4打數3安打1支全壘打2得分2打點，幫助洋聯明星隊5:5逼和央聯，大谷因而贏得該場MVP
  • 2016年 -- 07月24日，自7月10日對千葉羅德海洋隊先發登板但手指受傷後，首度登板投球，對歐力士隊中繼投球1局無失分，取得生涯首次中繼成功
  • 2016年 -- 09月13日，對歐力士猛牛隊糸井嘉男於三局上投出164km/h的速球，刷新自己保持的日職最快球速紀錄
  • 2016年 -- 09月28日，對埼玉西武獅隊完封9局僅被擊出1安，全場投出15次奪三振。達成生涯最長的9連勝與單季第10勝，繼2014年後完成生涯第二度「10勝、10轟」紀錄，也是史上第一次完成「10勝、100三振、20轟、100安」的球員，同時使火腿隊獲得該季洋聯優勝
  • 2016年 -- 10月16日，在對福岡軟銀鷹隊的高潮系列賽決勝輪第5戰擔任先發指定打擊，後於九局上取消指定打擊、登板救援1局無失分，完成生涯第一次救援成功（季後賽紀錄）。另外也對吉村裕基投出刷新個人暨聯盟最快的165km/h速球，並對本多雄一投出達151km/h速度的指叉球，終場火腿隊以7：4獲勝，晉級日本大賽
  • 2016年 -- 10月22日，日本大賽第1戰對戰廣島東洋鯉魚隊，先發投球6局11三振，被擊出2轟失3分，打擊則有3打數2安打，終場火腿1：5敗給廣島。大谷在日本大賽第一次登板投球就苦吞敗投，並終止自05月29日以來投球兼打擊「二刀流」場次的8連勝（含季後賽）
  • 2016年 -- 10月25日，日本大賽第3戰，十局下對大瀬良大地擊出再見安打，助火腿隊取得系列賽首勝。大谷也成為繼1958年稻尾和久、1986年工藤公康以來，第3位在日本一系列賽曾擔任先發投手又擊出再見安打的球員
  • 2016年 -- 11月12日，在「日本武士隊對荷蘭、墨西哥國家隊強化比賽」第3戰中，面對荷蘭隊投手Jair Jurrjens，擊出加入職棒後於國家隊的首支全壘打
  • 2016年 -- 11月13日，「日本武士隊對荷蘭、墨西哥國家隊強化比賽」第4戰，大谷於7局上半上場代打，擊出飛球進入東京巨蛋屋頂的縫隙而未落下，經現場主審張展榮召集裁判群討論後判定為場地規則二壘安打，也是繼2002年松井秀喜以來，擊出此種安打的第二人
  • 2016年 -- 季後獲投手與指定打擊最佳十人獎，為日職史上首度單季將複數座最佳十人獎頒予同位球員，並獲洋聯年度MVP與日本職業體育界最高榮譽的日本職業體育大獎
  • 2017年 -- 季前原入選第四屆世界棒球經典賽日本代表隊，但於02月03日因右腳傷勢未癒而宣佈退出
  • 2017年 -- 07月12日，該季首度以投手的身分先發出賽，對歐力士隊僅先發1.1局失4分吞敗
  • 2017年 -- 09月12日，先發對戰東北樂天金鷲隊，主投5.2局僅被擊出一支安打無失分，獲得本季首勝。此場比賽共有大聯盟16支球隊、32個球探前來觀戰
  • 2017年 -- 10月04日，對歐力士猛牛隊首度以第四棒‧先發投手的身份出賽，為1951年10月07日藤村富美男以來日職再度出現此紀錄。該場大谷投出9局10次奪三振的完封勝，打擊成績則為4打數1安打，跑回致勝分，終場火腿隊以3：0獲勝
  • 2017年 -- 10月12日，進行右腳關節三角骨去除手術，後於26日出院
  • 2017年 -- 11月07日，於秋訓期間首度表明有意進軍美職，並選擇CAA SPorts經紀公司作為其與美職球隊洽談合約的代理人
  • 2017年 -- 11月11日，經球團同意後召開記者會，正式宣佈挑戰大聯盟
  • 2017年 -- 12月09日，經過兩輪篩選與會談後，最終宣佈加盟洛杉磯天使隊並簽下小聯盟合約，10日舉行加盟記者會，背號則選擇「17」
  • 2018年 -- 02月01日，於今日準備搭機前往美國，同時也和日本航空簽下複數年贊助合約
  • 2018年 -- 03月29日，於開幕戰對奧克蘭運動家隊以先發第八棒‧指定打擊的打者身份上場，並在首打席即擊出美職生涯首安
  • 2018年 -- 04月01日，對奧克蘭運動家隊美職生涯首度登板，先發主投6局被擊出3安1轟失3分，並投出6次奪三振，以優質先發獲得美職生涯首場勝投
  • 2018年 -- 04月04日，對克里夫蘭印地安人隊對Josh Tomlin於一局下擊出旅美生涯首支全壘打，帶有三分打點，也是生涯首個主場打席就開轟
  • 2018年 -- 06月08日，因右手肘尺側副韌帶二級撕裂傷，採用自體血小板血漿（Platelet-rich plasma，簡稱PRP）注射治療，進入傷兵名單，預估至少需休養三周
  • 2018年 -- 07月03日，被移出傷兵名單，但因為傷勢因素，球隊決定先暫時只打擊，待傷勢轉好露再來評估重返先發輪值的可能性 
  • 2018年 -- 08月03日，對克里夫蘭印地安人隊擔當第三棒DH，5打數4安2轟3打點3得分1盜壘，完成美職新人年十轟的紀錄，亦同時達成美職生涯首度客場擊出全壘打、單場雙響砲及4安
  • 2018年 -- 09月05日，經檢查後發現右手肘韌帶出現新損傷，專家建議進行Tommy John手術，此外投球部分也確定本季不再登板；而面對遊騎兵隊單場雙響砲，以18轟（247個打數）追平城島健司在2006年所創下日籍選手菜鳥年單季18轟（506個打數）紀錄
  • 2018年 -- 09月07日，對白襪隊Carlos Rondon於三局上擊出三分砲，以19轟刷新日籍打者大聯盟菜鳥年最多轟紀錄
  • 2018年 -- 10月01日，接受Tommy John Surgery手術，由專擅運動傷害醫學的洛杉磯骨科名醫Neal ElAttrache操刀，成功完成手術。確認2019整個球季無法投球，僅能以DH出賽
  • 2018年 -- 11月12日，獲美國棒球記者協會BBWAA票選為美聯新人王，在30張選票中拿到25張第一名選票。並於次年1月26日親往紐約參加頒獎晚宴，首次以英文發表公開演說
  • 2019年 -- 03月11日，因去年奪下美聯新人王，加上投、打兩端均有出色表現，獲加薪10.5萬美元，2019年年薪為65萬美元，加薪幅度19.3%
  • 2019年 -- 06月14日，對坦帕灣光芒隊達成完全打擊，為旅美日本球員首位、亞洲球員第2位
  • 2019年 -- 09月13日，天使隊宣佈大谷為治療先天性髕骨分裂的問題而進行左膝手術，該季也提前結束
  • 2020年 -- 07月25日，對奧克蘭運動家隊以先發第三棒‧指定打擊的身份正式復出。此外由於該場賽事進入延長賽，實施本季限定的突破僵局制，大谷也在十局上按規定擔任二壘跑者，成為大聯盟實施該制度後首位突破僵局制跑者
  • 2021年 -- 02月09日，與球團簽下2年850萬、首年年薪300萬美元的合約，結束薪資仲裁程序。
  • 2021年 -- 04月22日，對德州遊騎兵隊Mike Foltynewicz於三局下擊出日美生涯通算第100支全壘打。
  • 2021年 -- 05月13日，對休士頓太空人隊擔任先發第一棒‧指定打擊，為大聯盟自1916年Ray Caldwell後，相隔105年再度出現先發投手登板後，隔日擔任球隊第一棒的調度。
  • 2021年 -- 07月03日，對巴爾的摩金鶯隊單場擊出雙響砲，達成單季第30轟，刷新Babe Ruth於1919年諦造的二刀流選手單季29轟的紀錄；此外也成為美聯首位在賽季前81場便達成30轟、10盜紀錄的球員。
  • 2021年 -- 07月05日，生涯首度入選美國職棒明星賽，為美職史上首位同時以投手及打者身份入選賽事的球員，並且也獲選參加全壘打大賽。
  • 2021年 -- 07月21日，美國最大體育用品公司Fanatics宣佈與其簽下獨家複數年合約，未來其簽名周邊以及實戰用品都將由Fanatics獨家販售，不過並不包含球員卡。
  • 2022年 -- 02月01日，美國電玩遊戲《MLB The Show》2022年版的封面人物，由大谷翔平擔任。
